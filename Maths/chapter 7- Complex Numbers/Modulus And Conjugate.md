
If $z = a + ib$, then:

- $|z| = \sqrt{a^2 + b^2} \quad (\text{Real number})$
    
- $\bar{z} = a - ib$

Note: all properties for polynomials work for complex numbers asw
	$(a+ib)^2=(a^2+(ib)^2+2iab)$


**Note:** 
	$Arg(Im(z)) = \pm \frac{\pi}{2}$
	$Arg(Re(z)) = 0\;or\; \pi$

$Arg(\bar{z}) = -Arg(z)$

# **Properties of Modulus:**


**Property 1:** $z \cdot \bar{z} = |z|^2$  
	Proof :$(a + ib)(a - ib) = a^2 + b^2$

**Property 2:** $|z_1 z_2| = |z_1| \cdot |z_2|$

**Property 3:** $\Big|\dfrac{z_1}{z_2}\Big| = \dfrac{|z_1|}{|z_2|}$

**Property 4:** $\overline{z_1 z_2} = \bar{z_1} , \bar{z_2}$

**Property 5:** $\overline{z_1 + z_2} = \bar{z_1} + \bar{z_2}$

**Property 6**:$z_1\bar{z_2} + \bar{z_1}z_2 = 2|z_1||z_2|\cos(θ_1-θ_2)$ ($\theta_{1}$and $\theta_{2}$ is Angle between $z_1$ & $z_2$)

**Property 7:**  $|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2|z_1||z_2|\cos(θ_1-θ_2)$

**Property 8:** $|z_1 - z_2|^2 = |z_1|^2 + |z_2|^2 - 2|z_1||z_2|\cos(θ_1-θ_2)$

## Multiplicative Inverse
what to multiply to get 1

Formula: $z^{-1} = \dfrac{\bar{z}}{|z|^2}$ (property 1)

1. For $z = 2 + 3i$:  
    $z^{-1} = \dfrac{2 - 3i}{(2+3i)(2-3i)} = \dfrac{2 - 3i}{13}$
    
2. For $z = -i$:  
    $z^{-1} = \dfrac{0 + i}{1} = i$
    
3. For $z = 3 - 3i$:  
    $z^{-1} = \dfrac{3 + 3i}{(3-3i)(3+3i)} = \dfrac{3 + 3i}{18} = \dfrac{1}{6} + \dfrac{1}{6}i$
