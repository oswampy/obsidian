When a line makes angle $\theta$ with the $x$-axis,  
$\theta$ is called an **argument (amplitude)**.

$\tan \theta = \dfrac{b}{a}, \quad \theta = \tan^{-1}\left(\dfrac{b}{a}\right)$
### Principle Values for Different Quadrants

-Quadrant 1
arg(z)=$\theta$

-Quadrant 2
arg(z)=$\pi-\theta$

-Quadrant 3  
arg(z)=$-\pi+\theta$
(argument is between $\pi$ and $-\pi$ so cant take full angle; its taken clockwise)

-Quadrant 4
arg(z)=$-\theta$

where $\theta$ = $\tan^{-1} \frac{b}{a}$

---

# Properties of Arguments

i) $arg(z_1z_2) = arg(z_1) + arg(z_2) + 2k\pi$

ii) $arg\left(\frac{z_1}{z_2}\right) = arg(z_1) - arg(z_2)$

iii) $arg\left(\frac{z_1}{z_2}\right) = arg(z_1) - arg(z_2) + 2k\pi$

iv) $arg\left(\frac{z_1}{z_2}\right) = 2arg z + 2k\pi$

v) $arg(z^n) = n arg(z) + 2k\pi$

vi) $arg\left(\frac{z_2}{z_1}\right) = 0$ $arg\left(\frac{z_1}{z_2}\right) = 2k\pi = 0$

vii) $arg z = -arg z = arg \frac{1}{z}$

viii) $arg(z-z̄) = \frac{\pm\pi}{2}$

ix) $arg(-z) = arg z \pm \pi$

xi) $arg(z) - arg(z^c) = \pm\pi$ 

---

### **Examples**

i) $\arg(1 + \sqrt{3}i)$  
$\theta = \tan^{-1} \sqrt{3}$  
$\theta = 60^\circ = \dfrac{\pi}{3}$

ii) $\arg(1 - \sqrt{3}i)$  
$\theta = \tan^{-1}(-\sqrt{3})$  
$\theta = -60^\circ = -\dfrac{\pi}{3}$
since the complex number is in the fourth quadrant (+,-) we measure the angle anti clockwise starting from origin.
so $\theta = 2\pi - \dfrac{\pi}{3} = \dfrac{5\pi}{3}$
