
A circle is a set of points in the plane which is equidistant from a fixed point called centre.

## **General Equation of a Circle**

$ax^2 + ay^2 + 2gx + 2fy + c = 0$

centre $$C= (-g, -f)$$

radius $$ r= \sqrt{g^2 + f^2 - c}$$


• while solving $g$, we make coeff of $x^2$ and $y^2$ as 1

• If $\ell$ circle $S \equiv 0$ and $\ell$ line $l \equiv 0$ intersect, then the equation of circle passing through points of origin intersection of circle and line is $S + \lambda l = 0$ where $\lambda$ is a parameter

## **Diametric form of a circle**


$(x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0$


## **Parametric form of a circle**


$x = h + r\cos\theta$ $c \equiv (h, k)$

$y = k + r\sin\theta$

$r\cos\theta = x - h$

$r\sin\theta = y - k$


## **Intercepts made on axes by a circle**


$S \equiv x^2 + y^2 + 2gx + 2fy + c = 0$

Eq of $x$ axis $y = 0$ (i)

$x^2 + 2gx + c = 0$ (ii)

let $x_1, x_2$ be root of eq (ii)

$x_1 + x_2 = -2g$, $x_1 x_2 = c$

$(x_2 - x_1)^2 = (x_2 + x_1)^2 - 4x_1 x_2$

$x_2 - x_1 = \frac{4g^2 - 4c}{x_2 - x_1}$

$x_2 - x_1 = 2\sqrt{g^2 - c}$

$x$-intercept $= 2\sqrt{g^2 - c}$

$y$-intercept $= 2\sqrt{f^2 - c}$

If $S = 0$ touches $x$-axis, $g^2 = c$

If $S \equiv 0$ touches $y$-axis, $f^2 = ac$


## **Position of a point**


Let any point $P(x_1, y_1)$

$S \equiv ax^2 + y^2 + 2gx + 2fy + c = 0$

$S_1 \equiv a x_1^2 + y_1^2 + 2g x_1 + 2f y_1 + c = 0$

If $S_1 = 0$, $P$ is inside circle

If $S_1 = 0$, $P$ is on circle

If $S_1 > 0$, $P$ is outside circle

## **Concentric Circles**


If center is same
