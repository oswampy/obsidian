## Number of functions
let X and Y be two finite sets having m and n elements, each element of m can be mapped to any element of n so total possibilities is $n^m$ .

## Domain, co-domain and Range
*domain*- set of all real numbers for which the function is defined.
*codomain*-set of all possible outputs
*range*-subset of codomain which contains the ACTUAL outputs

## Algebra of functions
Let f(x) and g(x) be two real and single valued functions, with domains $X_f$ and $X_g$ and ranges $Y_f$ and $Y_g$ . Let X =  $X_f$ $\bigcap$ $X_g$

1) Scalar multiplication of a function:(cf)(x) = cf(x) where c is a scalar. the new function has domain $X_f$
2) Addition/subtraction of functions: f$\pm$g(x) = f(x) $\pm$g(x) new function has domain X
3) Multiplication of functions: the product of function has domain X
4) Division of functions: new function has domain X, denominator $\ne$ 0,$\frac{f}{g}(x)$ = $\frac{f(x)}{g(x)}$ 
5) Equal functions: two functions f and g are equal iff:
	1) domain of g = domain of f
	2) range of g = range of f
	3) f(x) = g(x) $\forall$ x $\in$ common domain
