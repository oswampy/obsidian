## Types of Relations
1) Empty Relation:
	A relation R on a set A is said to be empty iff R = $\varnothing$
2) Universal Relation:
	A Set is said to be universal iff R = AxA
3) Identity Relation:
	A relation R is an identity relation on a if
		R = {a,a} $\forall$ a $\in$ A (denoted by $I_{A}$)
4) Inverse Relation:
	If R is a relation from set A to set B, then inverse relations to be $R^{-1}$ a relation between B to A Dom of $R^{-1}$ is range of R and vice versa.
5) Reflexive Relation:
	R is a relation and for every a$\in$ A (a,a) $\in$ R , it is said to be reflexive
6) Symmetric Relation:
	R is a relation in A (a,b) $\in$ imples (b,a) $\in$ R, is said to be symmetric
7) Transitive Relation:
	R is a relation in Am If (a,b) $\in$ R and (b,a )$\in$ R implies (a,c)$\in$ R then R is transitive
8) Equivalence Relation:
	if a relation is reflexive, symmetric and transitive it is equivalence
