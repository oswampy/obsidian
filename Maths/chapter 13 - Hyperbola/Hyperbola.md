# Eccentricity 

Reading from right to left (as the page is rotated):

**Eccentricity**

$e = \sqrt{1 + \frac{b^2}{a^2}}$ (Hyperbola, $e > 1$)

For $LR$

x coordinate of LR will be ae for L and L'
put that in gen eq of hyperbola and 

$e^2-\frac{y^2}{b^2}=1$
$b^2(e^2-1)=y^2$ (from eq of eccentricity)
$\frac{b^4}{a^2}=y^2$
$y=\frac{b^2}{a}$
total length is 2y so 
LR = $\frac{2b^2}{a}$
$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$


| Property                  | $Hyperbola ( \frac{x^2}{a^2}-\frac{y^2}{b^2}=1 )$ | $Conjugate Hyperbola ( \frac{y^2}{b^2}-\frac{x^2}{a^2}=1 )$ |
| ------------------------- | ------------------------------------------------- | ----------------------------------------------------------- |
| Center                    | ((0,0))                                           | ((0,0))                                                     |
| Transverse axis           | Along **x-axis**                                  | Along **y-axis**                                            |
| Length of transverse axis | (2a)                                              | (2b)                                                        |
| Conjugate axis            | Along **y-axis**                                  | Along **x-axis**                                            |
| Length of conjugate axis  | (2b)                                              | (2a)                                                        |
| Eccentricity              | $(e=\frac{c}{a})$                                 | $(e=\frac{c}{b})$                                           |
| Foci                      | $((\pm c,0))$                                     | $((0,\pm c))$                                               |
| Vertices                  | $((\pm a,0))$                                     | $((0,\pm b))$                                               |
| Latus rectum length       | $(\frac{2b^2}{a})$                                | $(\frac{2a^2}{b})$                                          |
