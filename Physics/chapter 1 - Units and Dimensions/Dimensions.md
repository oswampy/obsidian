Dimensions express derived quantities in terms of fundamental quantities

principle of homogeneity-
	Magnitude of physical quantities may be added/subtracted together only if they have the same dimensions
- if an equation is dimensionally correct it may not be a valid equation
	- if incorrect it is obviously invalid
- all forces and energies have same dimensions
- exponential powers and angle have no dimensions
## uses of dimensional analysis
- to establish relation between different physical quantities

- if we know the various factors on which  physical quantity depends then we can find a relation among different factors by using principle of homogeneity
- to convert across systems:
	- magnitude x unit = constant
	- n1u1 = n2u2
