## Types of vectors
- Unit Vector - Magnitude is 1 in any direction ($\hat{x}$)
- Null Vector(zero vector)- zero magnitude, no direction
- Equal Vector - Same magnitude and direction
- Parallel Vector - Same Direction
- Anti-Parallel - Opposite Direction
- Negative Vector - Same magnitude but opposite directions
- Concurrent - Same initial point
- Coplanar - In same planes
- Orthogonal - Perpendicular to each other
