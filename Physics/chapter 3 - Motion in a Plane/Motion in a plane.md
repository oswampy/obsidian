## Projectile Motion ----------------------
 - Acceleration is only due to gravity, path is the trajectory

### Time of flight
$s= ut+\frac{1}{2}at^2$
$s_{y}= u_{y}t+\frac{1}{2}a_{y}t^2$
$0 = u\sin \theta t + \frac{1}{2}(-g)t^2$ (the object will be only displaced in x axis so $s_y$ = 0)
$$t = \frac{2u\sin\theta}{g}$$
---
## Max Height
when finding max height, $v_{y} = 0$ because then only particle will be at max height and wont fall further:
$v^{2}-u^{2}=2as$
$0 - (u\sin \theta)^2=2as$
$2gs = u^2\sin^2\theta$
$$s=\frac{u^2\sin^2\theta}{2g}$$
---
## Horizontal Range
- Horizontal Range is the distance travelled by the object after motion
$s_{x} =u_{x}t+\frac{1}{2}a_{x}t^2$
$s_{x} =u\cos \theta\left[\frac{2u\sin\theta}{g} \right]+\frac{1}{2} 0t^2$  (t= $\frac{2u\sin\theta}{g}$, $a_x$= 0 because g is always in y direc)
$$s_{x} = \frac{u^{2}\sin2\theta}{g}$$
- when it says range is maximum, theta = 45 (because sin is max at 1 and 2x45 = 90)
# Trajectory --------------------------------

	$U_{x} = u\cos \theta$
	$U_{y} = u\sin \theta$

## Time of flight
$x(t) = U_xt + \frac{1}{2} a_xt^2$
$x = u\cos \theta t + 0$
$$t = \frac{x}{u\cos \theta}$$
---
## Max height
$y(t) = u_{y}t + \frac{1}{2}a_{y}t^2$
$y(t) = u\sin \theta t - \frac{1}{2}gt^2$
$y(t) = u\sin \theta\left[ \frac{x}{u\cos \theta} \right]  - \frac{1}{2}g\left[ \frac{x}{u\cos \theta} \right]^2$ (subbing t in)
$y = x\sin \frac{\theta}{\cos \theta} - \frac{1}{2} \frac{gx^2}{u^2\cos^2}$
$y=x\tan \theta\left( 1-\frac{gx}{2u^2\cos^2\theta \sin \theta} \right)$
$$y = x\tan \theta\left( 1-\frac{x}{R} \right)$$
in case a q gives $y = ax-bx^2$ then R is simply $\frac{a}{b}$
# Horizontal Projectile
-
	![[Horizontal_proj.excalidraw]]
	$U_{y} = 0$
	$U_{x} = u$
	$a_{x} = 0$
	$a_{y} = -g$
	$S_{x} = R$
	$S_{y} = h$

## Time of flight
$S_{y} = U_{y}t + \frac{1}{2} a_{y}t^2$
$-h= 0 - \frac{1}{2} gt^2$
$$t = \sqrt{ \frac{2h}{g} }$$
---
## Horizontal Range
$S_{x} = U_{x}t + \frac{1}{2} a_{x}t^2$
$R = Ut + \frac{1}{2} 0t^2$
$$R = u\sqrt{ \frac{2h}{g} }$$
# Inclined Plane Projection
-
	![[Inclined_Plane_Projectile.excalidraw]]
	$U_{x} = u\cos \theta$
	$U_{y} = u\sin \theta$
	$a_{x} = -g\sin \theta$
	$a_{y} = -g\cos \alpha$
	$S_{y} = 0$

## Time of flight
$S_{y} = U_{y}t + \frac{1}{2} a_{y}t^2$
$0 = u\sin \theta t - \frac{1}{2} g\cos \alpha t^2$
$\frac{1}{2} g\cos \alpha t^{2}= u\sin \theta t$
$$t = \frac{2u\sin\theta}{g\cos \alpha}$$
## Max height
$V_{y}^{2}=U_{y}^{2}+ 2a_{y}s$ (s is just max y height)
$0 = u^2\sin^{2} \theta-2g\cos \alpha y_{max}$   (velocity at highest point is 0)
$$y_{max} = \frac{u^2\sin^2\theta}{2g\cos \alpha}$$
