first step is finding n factor:

| Acids                    | Bases                     | Salts                         |
| ------------------------ | ------------------------- | ----------------------------- |
| n factor is the basicity | n factor is acidity       | Total charge on cation        |
| no of replacable H ions  | no of replacable OH atoms | Al2(SO4)3                     |
| ex - HCl, n =1           | ex- NaOH, n =1            | if 'salt' mentioned, n = 6    |
| -H2SO4, n = 2            | - Ca(OH)2, n =2           | if 'Al' atom mentioned, n = 3 |
$$1)\:Equivalent\;Mass =\frac{Molar\;mass}{n\;factor}$$
$$2)\:Gram\;equivalent = \frac{given\;mass}{equivalent\;mass}$$
$$3)\:Normality = \frac{Gram\;equivalent}{Volume\;of\;solution(L)}$$
   OR (only for mcqs)
$$normality=\frac{moles\;*n \;factor}{volume}$$

#Q if 0.98g of H2SO4 present in 500 ml solution, find N
A)
- n = 2
- $Eq = \frac{98}{2} => 49$
- $g.e = \frac{0.98}{49} \implies {0}.002$
- $N = \frac{0.002}{0.5} \implies \boxed{0.004}$
