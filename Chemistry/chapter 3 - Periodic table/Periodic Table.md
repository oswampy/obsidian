# some exceptions
| Trend                  | Expected          | Exception                     | Correct Order         |
| ---------------------- | ----------------- | ----------------------------- | --------------------- |
| Ionization Energy      | Increases →       | Be > B, N > O                 | Be > B; N > O         |
| Electron Gain Enthalpy | More negative ↑   | Cl > F                        | Cl > F > Br > I       |
| Electronegativity      | Increases →↑      | F highest, but Cl has more EA | F > O > N > C         |
| Atomic Radius          | Decreases →       | O > N                         | N < O                 |
| Ionic Radius           | Depends on charge | Al³⁺ < Mg²⁺ < Na⁺ < O²⁻ < F⁻  | according to z        |
| Metallic Character     | Increases ↓       | Ga < Al                       | B < Ga < Al < In < Tl |
| B.D.E                  | Decrease↓         | $F_{2}$ is very small         | Cl₂ > Br₂ > F₂ > I₂   |
