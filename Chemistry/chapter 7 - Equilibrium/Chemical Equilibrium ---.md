## Law of Mass Action

A + B ⇌ C + D

**ROF ∝ \[A]\[B]**  \[x] = Active mass/conc° ROB ∝ \[C] \[D]

Kf = Rate const. of forward rxn 
Kb(P) = Rate const. of backward rxn

Kf\[A]\[B] = Kb\[C]\[D]

**eq^m const Ke = Kf/Kb = [C][D]/[A][B] {Product/Reactant}**

**ex- 2A + B ⇌ C + 3D** **Ke = \[C]¹\[D]³/\[A]²\[B]¹**

## Two types of eq^m const (Ke)

**\[x] = molarity = η (Conc)**  
**\[x] = gaseous phase = P(partial pressure) [ηp]**

| Formula         | Kc          | Kp               |
| --------------- | ----------- | ---------------- |
| PV = nRT        | η = M       | partial pressure |
| R= 0.082 (1/12) | liquids/gas | gases Px,g       |

## Expression and Units for Kp and Kc

#### 1) CaCO₃ ⇌ CaO(s) + CO₂

**Kc = \[CaO]¹\[CO₂]¹/\[CaCO₃]¹ = \[CO₂]¹ (mole/L)^Δng** where Δng=1

**※ [ ] of solids is always taken as 1.**

**Kp = (PCO₂)(atm)¹**

---

#### 2) N₂O₄(g) ⇌ 2NO₂(g)

**Kc = \[NO₂]²/\[N₂O₄]¹ (mole/L)¹**

**Kp = (PNO₂)²/(PN₂O₄)¹ (atm)¹**

---

#### 3) H₂O(l) ⇌ H₂O(g)

**※ [ ] of a liquid with gas in equilibrium is taken as 1**

**Kc = \[H₂O]¹/(mole/L)¹** **Kp = (PH₂O)¹(atm)¹**


---

#### 4) Ca(s) + 2Ag⁺(aq) ⇌ Ca²⁺(aq) + 2Ag(s)

**Kc = [Ca²⁺]¹/[Ag⁺]² (mol/L)¹**

**Kp = not defined (no gaseous element)**

---

#### 5) CH₃COOH(aq) + H₂O(l) ⇌ CH₃COO⁻(aq) + H₃O⁺

**Kc = [H₃O⁺]¹[CH₃COO⁻]¹/[CH₃COOH]¹ (mole/L)**

**※ If H₂O is in liquid then [H₂O] = 1**

## Relation between Kp and Kc

**Kp = Kc(RT)^Δng**
#Derivation 

**eq - 2A(g) + 3B(l) + 2C(s) ⇌ D(g) + E(g)**

| Kc = \[CD]¹\[CE]¹/\[CA]^2 | Kp = (PD)¹(PE)¹/(PA)^2 |
| ------------------------- | ---------------------- |

**P = (n/v)RT ⇒ CRT**

**In Kp = (CᴅRTb)¹(CᴇRT)¹/(CᴀRT)²**

**Kp = \[(Cᴅ)¹(Cʙ)¹/(Cᴀ)¹] × \[RT]**

**Kp = Kc × (RT)^Δng \[R = 0.082 L/mol/K]**
