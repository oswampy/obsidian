
## Cyclic Alkanes


### Priority Rules:

- **Functional group > double or triple bond > More no. of C atom**
- Chain has 5 weak of tie, ring is prefix
- Higher priority because suffix

---


### Alkane vs Alkyne

**Rules for naming:**

1. Select longest C-C chain with max no. of double or triple bonds
    
2. Numbering in such a way double/triple bond lowest no.
    
3. Same priority for both bonds but substituent have < = / ≡ bond
    
4. Priority in Case of Tie (double>triple)

	However name is written alphabetically

### Functional Groups

**Atom or group of atoms which only determines chemical properties**

- **Highest Priority**

### Order of Priority:

- **Car** - Carboxylic acid / R-C-OH (oic acid)
- **S** - Sulphonic Acid / -SO₃H (Sulphonic acid)
- **An** - Acid anhydride / R-C-O-C-R(-oic anhydride)
- **Tro** - Ester / R-C-O-R (-oate)
- **Aiss** - Acid Halide / R-C-X (-oyl halide)
- **Ambassador** - Amide / R-C-NH₂ (-amide)
- **C** - Cyanide / R-C≡N (-nitrile)
- **I** - Isocyanide / R-N≡C (-isonitrile)
- **A** - Aldehyde / R-C-H (-al)
- **K** - Ketone / R-C-R (-one)
- **A** - Alcohol / R-OH (-ol)
- **T** - Thio Alcohol / R-SH (-thiol)
- **Am** - Amine / R-NH₂ (-amine)

## Special Suffixes:

1. **Carboxylic acid** - Carboxylic Acid
2. **Amide** - Carboxamide
3. **Cyanide** - Carbonitrile
4. **Aldehyde** - Carbaldehyde


### **Use of Special Suffix**

If the functional group’s carbon atom is **not part of the main chain**, we use a **special suffix**.

Example structure:

COOH - C - C - C - COOH

(with another COOH branch on the middle carbon)

If we **cannot include all functional groups in the longest chain**, we ignore them all.

Example:  
**propane-1,2,3-tricarboxylic acid**

---

## **Polyfunctional Compounds**

- Functional group **higher in priority** → becomes **suffix**.
    
- The **rest** of the functional groups become **prefixes (normal branches)**.
